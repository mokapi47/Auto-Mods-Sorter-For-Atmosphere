# Auto-Mods-Sorter-For-Atmosphere
A simple Python file that will simplify your life to sort all your game mods for your Switch under Atmosphere, very intuitive, you hardly need to do anything, just place your game archives (.zip .rar .7z) in the root of the Python file, it will sort them all correctly according to the game chosen, with the Title ID adapting accordingly.

### If you have any problems, you can simply open a terminal and add the --debug argument to the end; a debug file with all the event details will be created.
Enjoy !
